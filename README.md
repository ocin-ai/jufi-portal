# Jufi-Portal – 's Badische Schullädle

Arbeitsaufträge und Prozessbeschreibungen der Juniorenfirma (Berufskolleg II, kaufmännisch).

**Website:** https://ocin-ai.github.io/jufi-portal/

Statische Website, gebaut mit MkDocs Material. Jeder Push auf `main` wird automatisch veröffentlicht (GitHub Actions). Quelle der Prozessinhalte ist ein lokaler Projektordner; die Synchronisation erfolgt über `sync_prozesse.py` mit Sperrliste (keine internen Dateien, keine personenbezogenen Daten).
