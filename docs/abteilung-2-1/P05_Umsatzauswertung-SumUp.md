- [ ] Login SumUp (vgl. 00_Logins *(interne Datei, nicht im Portal)*).
- [ ] Selektion des gewünschten Berichts (Berichte → Verkaufsbericht → Zeitraum) → exportieren.
- [ ] Export der Transaktionsdaten im .csv-Format und speichern (Ordner: `05_Umsatzauswertung-SumUp/`).
- [ ] Neue Excel-Datei öffnen, .csv-Datei importieren (Reiter Daten → Daten abrufen → ...).
- [ ] Speicherung der Transaktionsdaten auf dem Laufwerk (Ordner: `05_Umsatzauswertung-SumUp/`).
- [ ] Pivot-Tabelle erstellen.
- [ ] Diagramme zur Visualisierung erstellen.
